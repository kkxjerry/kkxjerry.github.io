# GitHub Pages Site

This folder contains a GitHub Pages-ready personal research homepage for Guichao Zhou.

## Files

- `index.html`: main homepage
- `styles.css`: site styling
- `script.js`: menu and reveal interactions
- `assets/images/`: profile image, paper previews, framework figures, favicon
- `assets/papers/`: manuscript PDFs
- `assets/cv/`: resume PDF

## How to publish

1. Create a GitHub repository named `<your-github-username>.github.io`.
2. Copy the contents of this folder into the root of that repository.
3. Commit and push to the `main` branch.
4. In the repository settings, open `Pages`.
5. Under `Build and deployment`, choose `Deploy from a branch`.
6. Select the `main` branch and the `/ (root)` folder.
7. Wait a few minutes, then open `https://<your-github-username>.github.io`.

GitHub's current quickstart is here:

- https://docs.github.com/en/pages/quickstart
