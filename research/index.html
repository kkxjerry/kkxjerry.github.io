IEEE TMM 大修 Multimodal LLM

Boosting Few-Shot Classification with Training-Free Multimodal In-Context Learning

Yuwu Lu, Guichao Zhou, Wai Keung Wong, Zhihui Lai, Xuelong Li

这项工作关注如何在不做额外训练的情况下，把现成的 multimodal LLM 直接用于 few-shot classification。我们设计了 Training-Free Prompting Framework， 通过 direct 与 reference-guided 两类 prompting 策略，缓解 positional bias、 hallucination 与 support-query 融合不足的问题。

Direct Candidate Prompting 适用于通用物体识别任务，直接把输出约束到候选类别名集合。
Reference-Guided Prompting 适用于细粒度任务，引入 support visual references 与结构化指令。
Training-Free Inference 不修改模型参数，不增加额外训练，强调部署轻量与模型通用性。
8 标准 few-shot benchmark
99.49% MiniImageNet, 5-way 1-shot
96.32% CUB200, 5-way 1-shot
  • 基于 Qwen2.5-VL-7B-Instruct，在 8 个标准 few-shot benchmark 上完成验证。
  • 在 5-way 1-shot 设置下，MiniImageNet 达到 99.49%，CUB200 达到 96.32%。
  • 体现了从 multimodal prompting 研究走向轻量化 Agent / 应用系统的潜力。

摘要速览

该框架通过更结构化的任务输入，提升模型从一般识别到细粒度分类的多模态推理能力。 它保留了模型通用性，避免 fine-tuning 成本，并在完全 training-free 的设定下显著提升 few-shot classification 表现。

Pattern Recognition 在投 Vision Foundation Model

AFF: Align-Filter-Fuse Tokens with Mutual Nearest Neighbor Matching for Training-Free Few-Shot Classification

Yuwu Lu, Guichao Zhou, Wai Keung Wong, Zhihui Lai, Xuelong Li

这篇工作提出了一个基于冻结 DINOv3 backbone 的 training-free few-shot classification pipeline。AFF 通过 align、filter、fuse 重新组织局部 token，MNN reducer 则保留 双向一致的 patch 匹配关系，从而让最终评分更稳定。

Align 利用 CLS token 作为语义锚点，先把 patch token 拉向类别相关方向。
Filter + Fuse 筛除无关 patch，并把全局语义重新注入保留下来的局部 token。
MNN Reducer 只保留双向一致的 token 匹配关系，提升 dense matching 的鲁棒性。
96.59% MiniImageNet 1-shot
+0.80 pt 相较强 token baseline
8.4× 相较 CPEA 的吞吐提升
  • 在 frozen DINOv3 protocol 下，MiniImageNet 1-shot 达到 96.59%。
  • 相较强 token baseline 提升 0.80 个百分点。
  • 在文中报告设置下，推理吞吐约为 CPEA 的 8.4 倍。

摘要速览

该方法完全不引入额外训练，重点解决 token purification 与 robust dense matching。 它使用 CLS token 作为语义锚点，并通过 mutual nearest-neighbor consistency 筛选 匹配关系，从而提升低样本识别任务中的稳定性与效率。
